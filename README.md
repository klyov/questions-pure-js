# Questions on pure js

### Installing

#### Install node package

```
npm i
```

or

```
yarn
```

#### Launch of the project

```
npm run start
```

or

```
yarn start
```

#### Build project

```
npm run build
```

or

```
yarn build
```

#### Deploy production to Firebase Hosting

```
firebase login
```

```
firebase init
```

```
firebase deploy
```
